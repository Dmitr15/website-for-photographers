# Responsive business card website with a feedback form

### Frontend: HTML/CSS/JS and BOOTSTRAP
### Backend: pure php 8.4.5
### Data Base: MySql via phpMySqlAdmin

## Features:
#### 1. To send a message, the user must specify: name, email, and the message itself.
#### 2. When you reload the page, the message is not sent again.
#### 3. After the message is successfully sent, the form will be deactivated.
#### 4. Validation of filled fields occurs both on the client and server sides.

## Installation
### There are only two steps to run this website:
#### 1. Download the project to the desired directory on your computer
#### 2. Run php -S localhost:8080 on your terminal. Navigate to http://localhost:8080 to see the site.
